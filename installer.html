Set up Page Crawler

10 seconds. Once.

1Open notebooklm.google.com in Chrome and sign in. Close the tab.
2Open this extension's folder.
3Double-click PageCrawlerSetup.app.
4Click Install, then Always Allow at the Chrome Safe Storage prompt.

If macOS blocks the app

You'll see "PageCrawlerSetup.app cannot be opened because Apple cannot check it for malicious software." Normal for any small unsigned tool.

Fix: Open System Settings → Privacy & Security. Scroll to the bottom. Click Open Anyway. Enter your Mac password. Double-click the app again.

What the Keychain prompt is

The helper needs to read the Google session cookies from your normal Chrome profile. macOS protects those cookies with Keychain, so it asks once: "Page Crawler wants to use 'Chrome Safe Storage'." Click Always Allow. You won't see it again.

This is what replaces a separate Google sign-in. The tool reuses the Chrome session you're already in.
